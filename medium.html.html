    <div class="problem">
        <h3 class="title">Problem 25: Maximum Subarray (Passed)</h3>
        <p>
            You are given an array of ints and must return the greatest sum that can be formed by adding consecutive values
            in the original array. For [-2, 1, -3, 4, -1, 2, 1, -5, 4] 6 would be returned from [4, -1, 2, 1].
        </p>
        <p>
            To solve this problem I mae two int variables current_max = 0 and total_max = INT_MIN. I then looped through the
            array of ints. For each iteration I set current_max = max(num, num+current_max) then total_max =
            max(current_max, total_max). After the loop ended I returned total_max. This works as current_max will keep track
            of the max between starting from the current number or adding to what was previously the max. total_max will
            then keep track of the max between the current max found from the current number, or the overall max from the
            previous numbers.
        </p>
    </div>
    <div class="problem">
        <h3 class="title">Problem 26: Insert Interval (Passed)</h3>
        <p>
            You are given a vector of vectors containing two ints representing intervals (ex. [[1,2],[4,5]]) and are given
            a vector of two ints representing a new interval to add. You must return a vector of intervals with the new
            interval incorporated without any overlap. [[1,2],[4,5]] with [2,7] becomes [1,7].
        </p>
        <p>
            My solution involved hard coding a lot of situations with if statements that would check specific conditions which
            saved space but wasted time. The easier solution that I found involves creating a left vector and right vector.
            You have a start = newInterval[0] and end = newInterval[1]. You then iterate through the vector of intervals
            and add it to the left vector if start > interval[1] and add to the right vector if end < interval[0]. For the
            merging of intervals if start or end is in between you set start = min(start, interval[0]) and end =
            max(end, interval[1]). Then you combine the vectors together and return the resulting vector.
        </p>
    </div>
